Visit Example.com

Click the link which is a absolute link above to visit Example.com in a new tab. _blank is the target attribute value that opens the link in a new tab. It redirects the link to a new browsing context.

The rel="noreferrer" attribute is used for security and privacy reasons. It prevents the browser from sending the referrer information to the linked website.
